AI Chatbot SaaS
Platformu

İşletmelerin kendi markaları altında AI destekli müşteri destek ajanları oluşturup çoklu kanallarda yayınlayabildiği kapsamlı bir SaaS platformu.

🤖

Vercel AI SDK

Gerçek zamanlı streaming yanıtlar. GPT-4o, Claude 3.5 Sonnet, Claude 3 Haiku desteği.

🌐

Firecrawl Entegrasyonu

Otomatik website crawling, sitemap desteği, PDF/DOCX/TXT dosya işleme.

🔍

pgvector RAG Pipeline

Semantic search, 512 token chunk'lar, cosine similarity ile doküman arama.

📨

Omnichannel Inbox

WhatsApp, Instagram, Facebook, Slack, Telegram, Email kanalları tek ekranda.

⚡

BullMQ Queue Sistemi

Background job processing, crawl işlemleri, bildirimler, token loglama.

💳

Stripe Entegrasyonu

Abonelik yönetimi, 14 günlük deneme, token bazlı fiyatlandırma.

🎫

Token Yönetimi

Gerçek token sayımı, model bazlı çarpanlar (GPT-4o: 5x, Mini: 1x).

🔐

Clerk Auth

Multi-tenant workspace, takım üyesi davet, rol yönetimi.

Kurulum

# 1. Bağımlılıkları yükle
npm install --legacy-peer-deps

# 2. Environment değişkenlerini ayarla
cp .env.example .env.local

# 3. Veritabanını kur
npx prisma migrate dev
npx prisma generate

# 4. Geliştirme sunucusunu başlat
npm run dev

# 5. Worker'ları başlat (ayrı terminal)
npm run worker
# Docker ile çalıştırma
docker-compose up -d
